# Isolated Network Deployment

export default ({ children }) => <DocsWrapper>{children}</DocsWrapper>

## HELM and docker dependency

Bailo depends on HELM charts provided by Bitnami and docker images found in dockerhub.Therfore, for isolated network without internet connectivity, internal hosted registry is required for:

- MinIO version tag ~10.0.1 https://charts.bitnami.com/bitnami
- MongoDB version tag ~10.31.3 https://charts.bitnami.com/bitnami
- Registy version tag 2
- seldon/seldon-core-s2i-python37:1.10.0
- maildev/maildev:20.0.2
- nginxinc/nginx-unprivileged

## NPM

Internal hosted npm repository is required.

## Optional dependencies
Selinium-webdriver and Sharp can be removed for isolated network deployment.
```
npm remove selenium-webdriver sharp
```
